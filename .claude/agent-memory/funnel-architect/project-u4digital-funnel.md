u4digital funnel designed 2026-05-01. Application funnel for high-ticket AI advisory targeting digital experts (R$10-80K/month).

**Why:** Zero existing audience, R$200 test budget, WhatsApp-first market. Needs simplest viable funnel to validate before scaling.

**How to apply:** Value ladder is Bait (ads) → Frontend (free Raio-X Digital diagnostic) → Middle (R$3-8K implementation) → Backend (R$5-15K/month retainer). Key insight: the live agent demo during the diagnostic is the AHA moment that drives conversion. Three faces being tested (Moroni/Tiago/brand) to determine long-term positioning. Manual WhatsApp follow-up in phase 1, automate only after 3-5 sales validated.
