Wedding Design System delivered to `brain/assets/branding/casamento-moroni-daphine/DESIGN_SYSTEM.md` on 2026-05-01.

**Why:** Moroni requested a comprehensive design system covering color, typography, spacing, components, motion, photography, and print specs for the moroniedaphine/ wedding project.

**How to apply:** This is the single source of truth for all wedding project design decisions. designer-agent, vfx-motion-designer, and dev-agent should consume this file before producing any output for moroniedaphine/.

Key locked decisions:
- 3 colors only: burgundy #4A1619, white #FFFFFF, gold #D4AF37
- Fraunces variable (display, opsz 144 / SOFT 50) + Inter (UI only)
- No script fonts, no custom SVG, no border-radius on buttons
- Lenis lerp 0.08, minimum 600ms transitions, Armani easing curve
- Hero entry stagger ~2.4s total
- Parallax ratio 0.15 (not 0.5)
- Section alternation: burgundy (hero) > white (content) > burgundy (close)
