# Launch Methodology Rules — REIS IA

> Used by: launch-strategist, cmo-strategist, traffic-manager, funnel-architect
> Purpose: Standard operating rules for any launch executed through the REIS IA agent system

---

## 1. Launch Type Selection

Every launch MUST follow one of these 5 models. No hybrid or improvised structures.

| Model | Source | Best For | Knowledge File |
|-------|--------|----------|---------------|
| Lançamento Semente | Érico Rocha (FL) | First launch, validating PMF | formula-lancamento-framework.md |
| Lançamento Clássico | Érico Rocha (FL) | Scaling after Semente success | formula-lancamento-framework.md |
| Lançamento Pago Turbo | Léo Tabari | Repeatable weekly/monthly paid events | low-ticket-launch-playbook.md |
| LCP (2-Day Intensive) | William Baldan | High-energy live 2-day events | lancamento-pago-baldan.md |
| Paid Launch (Hybrid) | Tabari + FL | Combining paid ticket + classic CPL structure | Both playbooks |

### Selection criteria:
- First time launching? → **Semente**
- Already validated, want to scale? → **Clássico** or **Turbo**
- Want repeatable, automated model? → **Turbo**
- Want live, high-energy community event? → **LCP**
- Client has existing audience? → **Clássico**
- Client starting from zero? → **Semente** or **Turbo** (low-risk)

## 2. Mandatory Elements (All Models)

Every launch, regardless of model, MUST include:

1. **NAR (Nicho-Avatar-Roma)** documented before any creative work begins
2. **Offer with 6 components**: Name, Roma, Deliverables, Bonuses, Price, Guarantee
3. **Objection Tree**: Universal + Specific objections mapped and addressed in content
4. **Content Strategy**: Raiz + Nutella content supporting the launch thesis
5. **Metrics tracking**: CPL, show-up rate, conversion rate, revenue per lead, ROAS
6. **Post-launch debrief**: Mandatory analysis document after every launch

## 3. Campaign Rules

- ALL campaigns created in **PAUSED** status (per ads-safety.md)
- Budget changes above R$100/day require **human approval**
- Paid launches: ticket revenue must achieve **ROAS >= 1.0** before scaling
- Creative portfolio: minimum **10 creative variations** per Andromeda rules
- Never pause "low ROAS" awareness ads without checking if they feed conversion ads

## 4. Content Production Rules

- Launch content follows **Raiz/Nutella** system:
  - Raiz (30min+) published first, distributed with paid traffic
  - Nutella (short-form) derived from Raiz, for reach expansion
- All content must address at least one of the 4 Hormozi angles:
  - More Good, Less Bad, More Good for Others, Less Bad for Others
- CPL content (Classic Launch) must progressively weaken objections
- Event scripts follow the structure: Roma → Proof → Story → Content → Offer → Ingredients

## 5. Quality Gate

Before any launch goes live:
- [ ] NAR documented and approved by cmo-strategist
- [ ] Offer reviewed by offer-architect
- [ ] Funnel mapped by funnel-architect
- [ ] Copy through full Copy Squad pipeline (copywriter → humanizer → reviewer)
- [ ] Creatives reviewed by creative-strategist
- [ ] Campaigns created PAUSED by traffic-manager
- [ ] Human approval for activation
