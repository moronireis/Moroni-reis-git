1. Resumo do Projeto

Construção de um CRM personalizado para corretora de seguros, integrando WhatsApp (API oficial Meta 24/7), ERP Corp (Agia) e automações por mudança de etapa. Substitui a necessidade de Clint + iSpeed + WhatsApp Web fragmentados.

Escopo
  • Inbox WhatsApp unificado (3 números + grupos)
  • CRM com campos de seguros
  • N negócios por contato
  • Funis personalizáveis (aberto)
  • Automações por etapa
  • Avisos de parcela via WhatsApp
  • Campanhas em massa segmentadas
  • Variáveis em mensagens (#nome, #veículo, #período_dia)
  • Visualização e interação com grupos WhatsApp
  • Dashboard + relatórios (CSV/PDF)
  • Tracking de links por campanha
  • Multi-usuário com perfis
  • Integração Corp (API Agia)
  • Chatbot de primeiro atendimento
Cronograma
1
Semana 1-2: Infra + WhatsApp API
Conexão dos 3 números, webhook 24/7, inbox funcional
2
Semana 3-4: CRM + Funis + Campos
Kanban/Grade, campos seguros, múltiplos negócios
3
Semana 5-6: Automações + Corp
Motor de regras, integração API, disparos
4
Semana 7-8: Refinamento
Ajustes, treinamento, chatbot, go-live

7. Integração Corp (Agia) — Pontos Críticos

Dependência principal do projeto.

A integração com o Corp define o que conseguimos automatizar. Precisamos validar os endpoints ANTES de começar o desenvolvimento das automações (Fase 3). Se a API for limitada, adaptamos o escopo.
