## Compatible with Reis IA core system

These patterns work with dark mode default, Inter font, #4A90FF accent, geometric minimal aesthetic:

- **Dark botanical photo overlay** — desaturated photography under warm dark rgba overlay. Architectural/geometric photos replace botanical ones for Reis IA.
- **Ultra-light display headline** — font-weight 300 at display scale, negative letter-spacing. Inter 300 already in Reis IA system.
- **CSS-only luxury transitions** — hover scale(1.04), scroll fadeInUp translateY(24px), all at editorial easings. Stack-agnostic.
- **Warm neutral anchor palette** — ivory→charcoal scale usable in light sub-sections, print materials, or as contrast against Reis IA dark surfaces.
- **Device mockup floating group** — composition technique (desktop+laptop+phone+artefact), adaptable by swapping invitation card for a Reis IA product artefact. Needs art-director sign-off.
- **Circular text badge** — adaptable if hourglass motif replaces name arc text. Needs art-director sign-off.

## Restricted to moroniedaphine sub-project only

These patterns conflict with Reis IA brand rules (PROHIBITED: botanical motifs incompatible with geometric aesthetic; warm paper palettes incompatible with dark mode default; calligraphic script incompatible with Inter-only system):

- **Serif + script name treatment** — requires calligraphic script font; Fraunces already approved for moroniedaphine
- **Invitation card 5:7** — warm paper palette, flat corners, physical print aesthetic
- **Botanical shadow cast composition** — organic motifs, photography-based
- **Material surface simulation** — linen/parchment/marble textures; warm and tactile, conflicts with Reis IA architectural coldness

**Why:** Reis IA brand rules explicitly prohibit warm organic aesthetics. The geometric dark minimal system is incompatible with paper textures, calligraphic scripts, and botanical motifs. These belong exclusively to the wedding sub-project where Fraunces + burgundy/white/gold is the locked aesthetic.
