## Notes para o Designer

### Tokens Fixos (todos os estáticos)
- Background: #0A0A0A (padrão) ou #1E1E1E (secundário para elementos internos)
- Texto principal: #FFFFFF
- Texto secundário: rgba(255,255,255,0.55)
- Acento: #74C69E
- Verde escuro sutil: #0F1A15 (apenas para blocos de destaque positivo)
- Font: Inter (Google Fonts ou self-hosted)
- Sem bordas arredondadas acima de 4px nos elementos de dado
- Grain noise 3-5%: aplicar com filtro SVG ou CSS noise para warmth

### Dimensões
- Feed: 1080x1080px
- Stories: 1080x1920px — manter zona segura 250px superior e inferior (fora de interface)

### Regras de Typography
- Hierarquia: Black/ExtraBold para hooks, Regular/Light para supporting, SemiBold para labels/dados
- Nunca usar mais de 3 pesos tipográficos num mesmo estático
- Line height para hooks: 1.1 (tight)
- Letter spacing para dados grandes: -0.02em (leve compressão)

### O que nunca aparece
- Emojis
- Gradientes de cor
- Sombras coloridas (apenas sombras escuras suaves se necessário)
- Mais de 2 cores de texto por estático (branco + #74C69E)
- Logotipos de ferramentas de IA (ChatGPT, OpenAI, etc.)
- Fotos de pessoas
