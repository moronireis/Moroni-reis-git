## 1. Visual Theme & Atmosphere

Bottega Veneta's digital presence is a masterclass in luxury restraint. The interface operates on a strict black-and-white foundation with zero chromatic accent — no brand color punctuates the UI because the house's identity IS absence of ornamentation. Where other luxury brands lean on a signature color (Tiffany blue, Cartier red, Hermes orange), Bottega Veneta's signature is the craft itself: the intrecciato weave, the leather texture, the product photography. The website exists purely to frame these objects.

Typography is delivered through a custom proprietary serif face — `bottegaveneta-regular-webfont` — that carries the weight of the entire visual identity. It is used at weight 400 universally; there is no display-weight theatrics, no tracking manipulation, no typographic showmanship. The bold variant (`bottegaveneta-bold-webfont`) appears only in navigation and button contexts where functional emphasis demands it. This typographic discipline — one face, two weights, natural spacing — mirrors the house's philosophy that the work speaks without decoration.

Surfaces are flat. There are no shadows, no elevation layers, no glassmorphism, no gradients. Depth is communicated entirely through photography (which is allowed to dominate) and the thin gray borders (`#d8d8d8`) that quietly separate content zones. The system is intentionally anti-digital: it reads like a printed catalog translated to screen, where the paper is white, the ink is black, and the craft is in the making, not the presentation.

The single most distinctive visual choice is the absolute commitment to zero color accent in the interface. Every interactive element — links, buttons, navigation — is rendered in pure black `#000000`. Even the "green" states (`#009539` for availability) are functional necessities, not brand expressions. This creates a visual environment where product photography holds total chromatic authority.

**Key Characteristics:**
- Pure black `#000000` as both brand identity and UI action color — no separate accent
- Custom proprietary serif typeface at exclusively weight 400 (regular) and 700 (bold only for buttons/nav)
- Zero border-radius on all interactive elements — sharp rectangles everywhere
- No shadows, no elevation, no blur — intentionally flat
- Photography-dominant: UI recedes completely, imagery commands attention
- Monochrome gray scale from `#272727` through `#767676` to `#d8d8d8` — never saturated
- White surface with no warm or cool tint — pure `#ffffff`
- Minimal state differentiation: hover states rely on underlines and font-weight, not color shifts
- Navigation uses 12px bold uppercase — functional signposting, not decorative type
- Error red (`#eb0000`) is the only saturated color in the system, reserved for validation

## 3. Typography Rules

### Font Family

The primary typeface is `bottegaveneta-regular-webfont` — a custom proprietary serif commissioned for the house. It is self-hosted in woff2/woff/ttf/eot formats with `font-display: optional` (meaning it will not flash unstyled — if the font doesn't load instantly, the page renders without it). The bold variant `bottegaveneta-bold-webfont` is a separate face file, not a variable-weight axis. A personalization variant `bottegaveneta-regular-perso-webfont` exists with `font-display: swap` for monogramming/engraving contexts. No monospace companion exists in the brand system; `monospace` is used only for code/debug contexts. No OpenType features are explicitly declared.

### Principles

- The brand typeface is a serif — never substitute with a generic sans-serif like Arial or Helvetica for headings. The serif character IS the brand voice on screen.
- Weight 400 is the default for ALL text including headlines. Weight 700 appears ONLY in navigation and buttons — never in headings, body, or descriptive text.
- Zero letter-spacing everywhere. No tracking is applied at any size — the typeface is designed to set naturally without adjustment.
- Line-height is consistently 1.4 for body contexts and tightens to 1.15-1.2 only at display sizes (35px+). There is no decorative leading.
- The bold face (`bottegaveneta-bold-webfont`) is a separate font file, not a weight axis. Do not interpolate between weights.
- Font-display is `optional` for the regular face, meaning the page will render with the system fallback if the custom font takes longer than ~100ms to load. This reflects luxury confidence: the typography is important but the experience should never feel broken.

## 5. Layout Principles

### Spacing System

Base unit is 16px (1rem), with a scale built in increments of 4px. The system favors generous vertical spacing between sections (80-160px range) while keeping horizontal content padding moderate (16-24px on mobile, 60px on desktop).

Key spacing values from the CSS: 4px (0.25rem), 5px (0.3125rem), 8px (0.5rem), 10px (0.625rem), 15px (0.9375rem), 16px (1rem), 20px (1.25rem), 24px (1.5rem), 25px (1.5625rem), 27px (1.6875rem), 30px (1.875rem), 32px (2rem), 40px (2.5rem), 60px (3.75rem), 80px (5rem), 100px (6.25rem), 160px (10rem).

### Grid & Container

The layout uses a full-bleed approach where hero imagery and campaign content span the entire viewport width. Content containers max out at the browser width — there is no explicit `max-width` cap on product grids. Product listing grids use 2-column on mobile expanding to 3 or 4 columns on desktop, with zero gap between product tiles (the photographs tile edge-to-edge). Category navigation uses a slide-from-right panel that transforms `translateX(100%)` → `translateX(0)`.

### Whitespace Philosophy

Whitespace on Bottega Veneta's site is not a design choice — it is a material. The home page deploys full-viewport imagery sections separated by deliberate emptiness, creating a rhythm that mirrors the experience of walking through a physical Bottega Veneta boutique where products are spaced far apart on vast neutral surfaces. This whitespace communicates the same luxury signal as an empty gallery wall: the absence is intentional, and the absence itself costs money (the space could have been filled with more products, more text, more persuasion — the brand chooses not to). Section padding regularly exceeds 80px vertically, and product card grids leave no gap between tiles, creating a photographic mosaic effect that treats the grid as a single composed image rather than individual units.

### Border Radius Scale

- **0px** (none): Buttons, inputs, cards, navigation, tabs — the dominant radius across the entire system. This is the brand signature shape.
- **3px** (0.1875rem): Applied only to specific link contexts — a barely-perceptible softening
- **5px**: Applied to specific UI corners — rare
- **10px** (0.625rem): Applied to specific button variants and card contexts — infrequent
- **50%**: Circle — used only for round notification badges and close buttons

The system overwhelmingly defaults to 0px. Any border-radius above 0 is an exception, not a design pattern.

## 6. Depth & Elevation

| Level | Treatment | Use |
|-------|-----------|-----|
| Flat (default) | No shadow, no elevation | All components — cards, buttons, inputs, navigation |
| Border | `1px solid #d8d8d8` | Section dividers, form field borders, table lines |
| Overlay | `rgba(0,0,0,0.2)` background | Modal backdrops, drawer overlays |
| Inset | `inset 0 1px 3px #000` | Specific pressed-button states only |

The system is intentionally zero-elevation. Depth is communicated via border + surface contrast, not shadow. There are no ambient shadows, no card elevation, no floating elements. The modal overlay uses a simple `rgba(0,0,0,0.2)` backdrop — not a blur, not a gradient, just a flat semi-transparent wash.

### Shadow Philosophy

Bottega Veneta's digital system has no shadow vocabulary because shadow implies a layered interface — elements floating above a surface, creating a z-axis hierarchy of importance. The house rejects this metaphor. Every element exists on the same plane as every other element, as if printed on the same sheet of paper. This flatness is not a trend choice (the site predates the flat-design movement); it is a philosophical position: the product is real and dimensional, the interface is not. The interface should never try to simulate physicality — that is the product's domain.

## 7. Do's and Don'ts

### Do's

- **Do** use `bottegaveneta-regular-webfont` at weight 400 for all headings, body text, and descriptions — the regular weight IS the brand voice
- **Do** use `bottegaveneta-bold-webfont` at weight 700 exclusively for buttons and navigation — nowhere else
- **Do** use pure black `#000000` for all links — Bottega Veneta links are never blue, never underlined by default
- **Do** use 0px border-radius on all buttons and inputs — sharp rectangles are the brand shape vocabulary
- **Do** let product photography fill the full width of its container with no padding, rounded corners, or decorative framing
- **Do** use `#767676` for secondary text (prices, metadata, descriptions) and `#272727` for form labels
- **Do** use `1px solid #d8d8d8` for all dividers and borders — never thicker, never darker
- **Do** use `transition: all 0.2s linear` for micro-interactions — BV uses linear easing and 200ms duration as the standard

### Don'ts

- **Don't** use any chromatic accent color (blue, green, red, purple) as a brand element — BV's digital accent is black
- **Don't** apply border-radius greater than 0px to buttons, cards, or inputs — rounded corners read as SaaS, not luxury
- **Don't** use weight 300 (light) for any text — BV's lightest weight is 400. Light weights read as "tech-elegant," not craft-confident
- **Don't** add box-shadow to cards or containers — the system is intentionally flat, shadows break the printed-catalog metaphor
- **Don't** use backdrop-filter or glassmorphism — BV's surfaces are solid white, not translucent
- **Don't** apply letter-spacing to any text at any size — the proprietary typeface sets naturally without tracking adjustments
- **Don't** use gradient fills on any element — no gradient backgrounds, no gradient text, no gradient borders
- **Don't** use `#eb0000` (red) for anything other than error states and sale indicators — it is not a brand color
- **Don't** use bounce, spring, or elastic easing — BV uses linear easing; even ease-out is rare. The motion is clinical, not playful
- **Don't** add decorative icons or emoji to the UI — BV uses two icon fonts (`icons`, `masked-text-webfont`) for functional icons only (arrows, close, search, bag)

## 8. Responsive Behavior

### Breakpoints

| Name | Width | Key Changes |
|------|-------|-------------|
| Mobile | < 768px (48em) | Single-column layout, hamburger nav, full-width product tiles in 2-col grid |
| Tablet | 768px - 1024px | Menu expands, product grid to 3 columns, navigation reveals horizontal |
| Desktop | > 1024px | Full horizontal navigation, product grid to 3-4 columns, campaign sections at full-bleed |

### Touch Targets

Button padding of `15px 0` with full-width behavior ensures adequate touch targets on mobile. Navigation hamburger menu and close buttons use ample hit areas. The slide-from-right panel pattern (`translateX(100%)` entry) provides a native-feeling navigation drawer on mobile.

### Collapsing Strategy

- **Typography**: Display sizes (57px hero) scale down, but the custom serif remains at all breakpoints. Font sizes generally maintain their rem values, relying on viewport-width imagery to create responsive hierarchy rather than font-size breakpoint jumps.
- **Grid**: Product grids collapse from 4 → 3 → 2 columns. Hero sections maintain full-bleed at all sizes.
- **Navigation**: Horizontal category bar on desktop collapses to a full-screen slide-from-right overlay on mobile with stacked category lists.
- **Campaign content**: Full-viewport campaign images maintain aspect ratio and bleed edge-to-edge regardless of viewport width.

### Image Behavior

Product images are the centerpiece of the responsive system. They maintain their aspect ratio and fill available width completely — no padding, no margins, no rounded corners. Campaign hero images use full-viewport-height treatments. Product grid images are uniformly sized within their column structure, creating a perfectly aligned photographic mosaic.
