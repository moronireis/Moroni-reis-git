08 — Nota importante

Histórico de conversas

Ponto levantado pelo Cláudio na reunião de 08/06.

Cláudio (08/06): “Um ponto fundamental é o histórico de conversa. Vai ficando sempre o histórico, data, o que foi conversado, o que foi proposto. Se não tiver nessa etapa, coloquem já para a próxima.”
Status: O histórico completo de conversas por cliente (com data, responsável e contexto) entra na Fase 2. Na Fase 1, o banco de dados já registra disparos realizados (quem recebeu, quando, qual mensagem). O histórico detalhado por cliente será a base da Fase 2.
Contexto: Cláudio mencionou que a equipe pode migrar para um modelo de atendimento por representada (não por cliente). Isso significa 5-6 pessoas diferentes falando com o mesmo cliente. O histórico é crítico para que ninguém perca o fio da conversa.
